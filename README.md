## ⚡️ 사용 라이브러리

- Spring Boot : 3.1.3
- Thymeleaf
- JPA

## ✅ 구현 내용

#### 1. 관리자

1. 학교 정보 검색
2. 학교 정보 상세 보기
3. 학교 등록/수정/삭제
4. 학교 기본 정보 CSV 파일 관리 및 일괄 적용
   
#### 2. 유저

1. 학교 검색 (초/중/고)
2. 주변 학교 검색
3. 학교 정보 상세 보기 및 주변 날씨 확인
